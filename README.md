# SCT_WD_01
Responsive Landing Page with Interactive Navigation Menu | HTML, CSS &amp; JavaScript | Fixed Navbar | Scroll Effects | Hover Animations | Fully Responsive UI

# 📖 Project Overview

This project is a **Responsive Landing Page** developed using **HTML5, CSS3, and JavaScript**. The landing page includes a **fixed navigation menu** that remains visible while scrolling and dynamically changes its appearance based on user interaction.

The primary objective of this project is to create a visually appealing, user-friendly, and fully responsive website that provides smooth navigation across different sections. The navigation menu changes color or style while scrolling and also includes hover animations to improve user experience.

This project was developed as **Task 01** for the **SkillCraft Technology Web Development Internship**.

---

# 🎯 Project Objectives

The main objectives of this project are:

- Create a professional landing page.
- Build a responsive layout for all devices.
- Implement a fixed navigation bar.
- Change navbar appearance while scrolling.
- Add hover effects to navigation items.
- Improve user interaction using JavaScript.
- Practice responsive web design concepts.
